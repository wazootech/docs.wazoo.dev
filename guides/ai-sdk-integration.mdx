Wazoo provides first-class integration with the [Vercel AI SDK](https://sdk.vercel.ai) for building agentic applications.

## Installation

```bash
deno add jsr:@worlds/client npm:ai
```

## Agent with hybrid search

```typescript
import { Client } from "@worlds/client";
import { generateText } from "ai";

const client = new Client();

// Search the knowledge graph
const results = await client.search("What is a knowledge graph?");

// Use results in an AI prompt
const response = await generateText({
  model: myModel,
  prompt: `Answer based on: ${results.map(r => r.text).join("\n")}`,
});
```

## Tool-based agents

```typescript
import { Client } from "@worlds/client";
import { tool } from "ai";
import { z } from "zod";

const client = new Client();

const searchGraph = tool({
  description: "Search the knowledge graph",
  parameters: z.object({ query: z.string() }),
  execute: async ({ query }) => {
    return client.search(query);
  },
});
```

## Available tools

The `@worlds/client/ai-sdk` module exports pre-built tools for AI SDK agents:

- `searchGraphTool` — Hybrid search over the knowledge graph.
- `queryGraphTool` — SPARQL query execution.
