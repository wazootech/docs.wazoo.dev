---

The `@worlds/denokv` package provides persistent graph storage using Deno KV — a built-in key-value store in the Deno runtime.

## Installation

```bash
deno add jsr:@worlds/denokv
```

## Usage

```typescript
import { createDenokvClient } from "@worlds/denokv";

const client = await createDenokvClient();

// Now use client.search(), client.query(), client.import()...
```

## Features

- **Keyword FTS** — Full-text search via Deno KV.
- **Native Deno integration** — No external database process.
- **Serverless ready** — Deploy on Deno Deploy.
