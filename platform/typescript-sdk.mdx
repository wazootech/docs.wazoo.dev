title: "TypeScript SDK"
description: "Use @wazoo/client to manage Wazoo platform resources."

`@wazoo/client` is the generated TypeScript SDK for the Wazoo Platform API. It
covers management-plane operations: users, Worlds, platform tokens, World
data-plane tokens, usage, limits, and billing.

For data-plane graph operations (search, SPARQL, import, export), use
[`@worlds/client`](https://jsr.io/@worlds/client).

## Install

```bash
npx jsr add @wazoo/client
```

## Create a client

```typescript
import { createClient } from "@wazoo/client";

const client = createClient({
  baseUrl: "https://api.wazoo.dev",
  auth: process.env.WAZOO_PLATFORM_TOKEN,
});
```

The token must be a platform token with the `wzp_` prefix. The client is a
`@hey-api/client-fetch` instance — use `throwOnError: true` if you prefer
exceptions over discriminated unions.

## Worlds

```typescript
import {
  listWorlds,
  createWorld,
  getWorld,
  updateWorld,
  deleteWorld,
} from "@wazoo/client";

// List active worlds
const { data } = await listWorlds({ client });
console.log(data?.worlds);

// Create a world
const created = await createWorld({
  client,
  body: {
    worldId: "support-knowledge",
    world: { displayName: "Support Knowledge", region: "auto" },
  },
});

// Get a world by ID
const world = await getWorld({
  client,
  path: { worldId: "support-knowledge" },
});

// Update display name
await updateWorld({
  client,
  path: { worldId: "support-knowledge" },
  body: {
    updateMask: "displayName",
    world: { displayName: "Support Knowledge Base" },
  },
});

// Soft-delete (recoverable for 30 days)
await deleteWorld({ client, path: { worldId: "support-knowledge" } });
```

Creating a platform world provisions a per-World Turso database. Use
`worlds-api.wazoo.dev` or `@worlds/client` for graph data operations inside that
world.

## Platform API tokens

```typescript
import {
  listPlatformTokens,
  createPlatformToken,
  deletePlatformToken,
} from "@wazoo/client";

// List tokens
const { data } = await listPlatformTokens({ client });

// Create a token (secret shown once)
const created = await createPlatformToken({
  client,
  body: {
    name: "my-read-token",
    scope: "users.read worlds.read usage.read",
  },
});
console.log(created.data?.token); // wzp_...

// Revoke a token by name
await deletePlatformToken({ client, path: { tokenName: "my-read-token" } });
```

## World auth tokens

World data-plane tokens use the `wzw_` prefix. They are scoped to a single
namespace and optionally a single world.

```typescript
import {
  listWorldTokens,
  createWorldToken,
  deleteWorldToken,
} from "@wazoo/client";

const { data } = await listWorldTokens({
  client,
  path: { worldId: "support-knowledge" },
});

const created = await createWorldToken({
  client,
  path: { worldId: "support-knowledge" },
});
console.log(created.data?.token); // one-time secret

await deleteWorldToken({
  client,
  path: { worldId: "support-knowledge", tokenUid: created.data!.token.uid },
});
```

## Usage and limits

```typescript
import { getWorldUsage, getWorldLimits } from "@wazoo/client";

const usage = await getWorldUsage({
  client,
  path: { worldId: "support-knowledge" },
});
console.log(usage.data?.usage.total);

const limits = await getWorldLimits({
  client,
  path: { worldId: "support-knowledge" },
});
```

## Billing

```typescript
import { getWorldBilling } from "@wazoo/client";

const billing = await getWorldBilling({
  client,
  path: { worldId: "support-knowledge" },
});
console.log(billing.data?.billing.state);
```

## Error handling

The SDK returns a discriminated union by default:

```typescript
const r = await listWorlds({ client });
if (r.error) {
  console.error(r.error.error.code, r.error.error.message);
} else {
  console.log(r.data?.worlds);
}
```

Set `throwOnError: true` on the client for exception-based handling.

## Runtime

The SDK is a fetch-based ESM package published on JSR. It runs in any modern
JavaScript runtime that provides `fetch`: Node.js, Cloudflare Workers, Deno,
Bun, and browsers.
