Wazoo's SPARQL engine is powered by [Comunica](https://comunica.dev), a modular SPARQL query engine for JavaScript.

## Query syntax

Supported SPARQL 1.1 features:

- `SELECT` queries with `?variables`
- `CONSTRUCT` for graph transformation
- `ASK` for boolean checks
- `DESCRIBE` for resource exploration
- `GRAPH` for named graph scoping
- `FILTER`, `OPTIONAL`, `UNION`, `ORDER BY`, `LIMIT`, `OFFSET`

## Engine interface

```typescript
interface SparqlEngineInterface {
  query(sparql: string): Promise<SparqlResponse>;
}
```
