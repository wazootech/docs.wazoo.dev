---
title: "update"
description: "Modify world metadata"
---

Updates the high-level metadata assigned to a World. Note that this does _not_
modify the graph data inside the World; for that, use the [sparql](./sparql.mdx)
command.

<ParamField path="--world" type="string" required>
  The public World ID or canonical resource name. *Alias:* `-w`
</ParamField>

<ParamField path="--display-name" type="string">
  A new mutable human-readable name for the World.
</ParamField>

<ParamField path="--state" type="string">
  Set the lifecycle state to `ACTIVE` or `SUSPENDED`.
</ParamField>

### Usage

```bash
worlds update --world my-world --display-name "Renamed World"
```
