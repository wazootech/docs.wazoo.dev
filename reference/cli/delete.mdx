---
title: "delete"
description: "Delete a graph and its indices"
---

Soft-deletes a World and starts its retention window. Direct get can still return
the retained World until purge, and `:undelete` can restore it while
`restorable = true`.

<ParamField path="--world" type="string" required>
  The public World ID or canonical resource name. *Alias:* `-w`
</ParamField>

### Usage

```bash
worlds delete --world my-world
```
