---
title: "create"
description: "Create a new knowledge graph context"
---

The `create` command requests a new persistent World instance on the server.

<ParamField path="--world-id" type="string" required>
  The immutable public World ID. It must match `^[a-z][a-z0-9-]{2,62}$`.
</ParamField>

<ParamField path="--display-name" type="string" required>
  The mutable human-readable title of your World.
</ParamField>

<ParamField path="--region" type="string">
  Optional hosted region hint. Defaults to `auto`.
</ParamField>

### Usage

```bash
worlds create --world-id my-world --display-name "My First World"
```
