The [worlds-client-evals](https://github.com/wazootech/worlds-client-evals) repository provides a deterministic assertion test framework for Wazoo agent tooling.

## Purpose

- Validate that AI agents produce correct SPARQL queries.
- Test search-then-SPARQL multi-hop patterns.
- Ensure tool descriptions and system prompts produce expected agent behavior.

## Key features

- Pre-built evaluation scenarios with known ground truth.
- Assertion-based testing of agent outputs.
- Integration with CI/CD pipelines.
