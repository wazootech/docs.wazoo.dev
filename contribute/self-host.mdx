## Current status

The current private deployment direction includes Docker Compose, private
networking with Tailscale or Cloudflare Tunnel, and hosted platform management
surfaces where appropriate.

The exact setup depends on which pieces you run yourself:

- Worlds data services and storage.
- Platform metadata services.
- Network ingress and access control.
- Token issuance and rotation.
