## Create a World

Worlds is the context engine for agents. Start with the hosted API when you want
managed storage, or use the same TypeScript SDK locally when you want embedded
memory during development.

### Prerequisites

1. Sign up at the [Worlds Console](https://console.wazoo.dev).
2. Generate an API key.
3. Save it as `WORLDS_API_KEY`.

### Install

<CodeGroup>
```bash Deno
deno add jsr:@wazoo/worlds-sdk
```

```bash npm
npx jsr add @wazoo/worlds-sdk
```

```bash Bun
bunx jsr add @wazoo/worlds-sdk
```

</CodeGroup>

### Run a graph loop

```typescript index.ts
import { Worlds } from "@wazoo/worlds-sdk";

const worlds = new Worlds({
  apiKey: process.env.WORLDS_API_KEY!,
  baseUrl: "https://api.wazoo.dev",
});

await worlds.create({
  slug: "quickstart-cloud",
  label: "My First World",
});

await worlds.import(
  "quickstart-cloud",
  `@prefix user: <https://example.org/user/> .
   @prefix wazoo: <https://wazoo.dev/#> .

   user:person wazoo:worksFor wazoo:organization .`,
  { contentType: "text/turtle" },
);

const result = await worlds.sparql(
  "quickstart-cloud",
  `PREFIX wazoo: <https://wazoo.dev/#>
   SELECT ?organization WHERE {
     <https://example.org/user/person> wazoo:worksFor ?organization
   }`,
);

console.log(JSON.stringify(result, null, 2));
```

For local or test workflows, use `LocalWorlds` from the same SDK and switch to
the hosted `Worlds` client when you need managed persistence.

## Validate a Markdown wiki

The Wiki toolchain turns a folder of Markdown into a validated, queryable,
publishable wiki.

```bash
pip install wazootech-wiki
wiki init
wiki check --strict
wiki serve --watch
```

Use this path for Obsidian vaults, LLM-maintained knowledge bases, internal
wikis, or static sites that need graph integrity.

Next: [Wiki toolchain](/projects/wiki) and
[Wiki templates](/projects/wiki/templates).

## Parse semantic Markdown

Linked Markdown extracts typed semantic frontmatter from Markdown.

<CodeGroup>
```bash Python
pip install linked-markdown
```

```bash Deno
deno add jsr:@wazoo/linked-markdown
```

</CodeGroup>

```typescript
import { extract } from "@wazoo/linked-markdown";

const result = extract(`---
{
  "@id": "https://example.org/docs/alice",
  "@type": "schema:Person"
}
---

# Alice
`);

console.log(result.attrs);
```

Next: [Linked Markdown](/projects/linked-markdown).

## Target portable memory

MemSDK gives your application a Supermemory-compatible memory interface that can
be implemented by multiple backends.

```bash
npm install github:wazootech/memsdk
```

```typescript
import type { SupermemoryInterface } from "memsdk";

async function remember(client: SupermemoryInterface) {
  await client.add({
    content: "The user prefers graph-backed memory.",
    containerTag: "user_123",
  });

  return client.search.documents({
    q: "graph-backed memory",
    containerTag: "user_123",
  });
}
```
